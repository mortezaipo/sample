# samplecmd
Prepare sample command usage

Simplecmd prepares examples about your specific commands.

### Example

```bash
$ samplecmd grep
```

### Arguments
| Description                      | Argument | Value  |
| -------------------------------- | -------- | ------ |
| Show command details (if it has) | -d       |        |
| Show more info about command     | -m       |        |
| Limit results                    | -l       | number |
